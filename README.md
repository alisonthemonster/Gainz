# Gainz :muscle: :trophy:
###### A weightlifting that projects future workouts based on your current progress, and allows easy reference to past workout history.

Actively helps the user make progress by pushing them to keep adding on weight, instead of only passively tracking past progress.

Simplicity is key - trying to handle one problem well instead of a couple of problems poorly.
